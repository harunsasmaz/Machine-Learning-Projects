# Machine Learning Projects

Koc University ENGR 421 Fall 2020 Course Projects

### Requirements

> Currently using MacOS Catalina 10.15.4

#### Environment

> Python 3.7
> PIP
> Jupyter Notebook
> R
> RStudio (Preferably)

#### Libraries

> NumPy
> Pandas
> matplotlib
> scikit-learn

> MASS (for R)

### Running

For python

```bash
cd <directory of .ipynb file>
jupyter notebook
```

Then, run cells one by one.

For R

> Open RStudio
> Set working directory
> Run source code
